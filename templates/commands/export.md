# /export - 多平台导出系统 📤

> **核心理念**: 将课程内容导出到不同平台的格式
> **目标**: 生成Notion/飞书/网易云课堂等平台可导入的文件

---

## 第一步: 运行脚本获取状态 ⚠️ 必须执行

```bash
bash scripts/bash/export.sh
```

脚本返回:
- `exports_dir`: 导出目录
- `exportable_content`: 可导出的内容统计
  - `outline`: 是否有大纲
  - `chapters`: 是否有章节内容
  - `exercises`: 是否有练习题
  - `scripts`: 是否有视频脚本

## 导出格式详解

### Notion Database 格式
```json
{
  "database_properties": {
    "名称": { "type": "title" },
    "章节": { "type": "number" },
    "时长": { "type": "text" },
    "类型": { "type": "select", "options": ["理论", "实践", "案例"]}
  },
  "pages": [...]
}
```

### 飞书文档格式
- Markdown格式,支持飞书的富文本语法
- 包含目录、正文、练习题

### 网易云课堂CSV格式
```csv
章节号,章节名称,课时号,课时名称,时长,类型
1,Python基础,1.1,Python简介,15分钟,理论课
```
